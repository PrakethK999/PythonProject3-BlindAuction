# Secret Auction 🔨
A silent auction program built with Python.

## What it does

* Accepts bids from multiple users one at a time
* Clears the screen between bidders to keep bids secret
* Stores each name and bid in a dictionary
* Automatically finds and announces the highest bidder

## What I learned

* Dictionaries and key-value pair storage
* While loops with continue and break
* Iterating over a dictionary to find a maximum value
* Screen clearing for a better user experience

## How to run

1. Clone the repository
2. Run `python Blind Auction.py`
3. Enter each bidder's name and bid, repeat until all bids are in
